# vue-x6-flow

找了一圈流程图和dag图插件，发现每一个的功能都太弱了没办法实现 运维可视化需求，所以自己花心思撸了一个出来。 
**基于antv-x6 vue版本**
vue2.x+ element +antv-x6 
适用于流程图-dag图

有以下功能内容：
1、拖拽布局
2、右键菜单
3、保存
4、缩放
5、多选
6、快捷键 cv  cx  cz（重做）
7、连线判断
8、配置属性
9、动态节点流向动画
其余功能可在antv-x6文档自行添加

## 文档地址
https://x6.antv.vision/zh/docs/api/graph/graph
